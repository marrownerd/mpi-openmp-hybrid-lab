# 3D Poisson Equation Solver (Hybrid MPI + OpenMP)

Параллельная реализация метода Якоби для решения уравнения Пуассона (Helmholtz type) в трехмерной области.
Лабораторная работа 2.

## Постановка задачи
Решается уравнение Пуассона с правой частью
Граничные условия 1-го рода.

## Реализация
*   **Язык:** C++
*   **Параллелизм:** Гибридная модель MPI + OpenMP.
*   **Декомпозиция:** 1D (Slab decomposition) вдоль оси X.
*   **Оптимизация:** Асинхронный обмен границами (`MPI_Isend`, `MPI_Irecv`). Вычисление внутренних точек области происходит во время передачи данных граничных слоев.

### Требования
*   Компилятор C++ (g++)
*   Библиотека MPI (OpenMPI, MPICH)
